# GitHub Portfolio

Portafolio personal estatico con estetica Matrix/Mr. Robot, pensado para enlazar desde LinkedIn o publicarlo con GitHub Pages.

## Como personalizarlo

1. Abre `index.html`.
2. Cambia `Luis Leonel Romero` por tu nombre completo, si luego quieres usar otro formato.
3. Los enlaces de GitHub y GitLab ya apuntan a `Leonel59`.
4. Cambia `tu-correo@ejemplo.com` por tu correo profesional.
5. Ajusta los proyectos destacados con tus repositorios reales.

## Como verlo

Abre `index.html` directamente en tu navegador. No necesita instalar dependencias.

## Como publicarlo en GitHub Pages

1. Crea un repositorio en GitHub, por ejemplo `github-portfolio`.
2. Sube estos archivos: `index.html`, `styles.css`, `script.js` y `README.md`.
3. En GitHub entra a `Settings > Pages`.
4. En `Build and deployment`, selecciona `Deploy from a branch`.
5. Elige la rama `main` y la carpeta `/root`.
6. Copia la URL publicada y pegala en tu perfil de LinkedIn.

## Tip para perfil de GitHub

Si quieres que aparezca en tu perfil principal de GitHub, crea un repositorio con el mismo nombre que tu usuario de GitHub y agrega un `README.md` con un enlace a este portafolio publicado.
